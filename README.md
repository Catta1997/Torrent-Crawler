How to:

    - Install Python3

Terminal:

    - install `requirements.txt` (`pip3 install -r requirements.txt`)
    - run `python3 torrent.py` (support for argv, `python3 torrent.py name`)

GUI:

    - install `requirements_gui.txt`  (`pip3 install -r requirements_gui.txt`)
    - run `python3 gui.py`
